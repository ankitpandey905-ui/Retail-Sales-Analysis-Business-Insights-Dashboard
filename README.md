# Retail-Sales-Analysis-Business-Insights-Dashboard
The retail company wants to better understand its sales performance and identify opportunities for improvement.
🔹 Project Overview

This project focuses on analyzing sales, profit, and discount data to extract meaningful business insights. The analysis was performed using Python for data cleaning and exploratory data analysis (EDA), and Power BI was used to build an interactive dashboard.

🔹 Objectives
Understand sales distribution and trends
Analyze category-wise and region-wise performance
Evaluate the impact of discount on profit
Identify high-performing and low-performing products
Provide actionable business recommendations
🔹 Tools & Technologies Used
Python (Pandas, NumPy)
Matplotlib & Seaborn (for visualization)
Microsoft Power BI Desktop
DAX (for calculated measures)
🔹 Data Cleaning Steps
Handled missing values
Removed duplicate records
Corrected data types
Formatted date columns
🔹 Exploratory Data Analysis (EDA)
Sales distribution analysis
Category-wise and region-wise performance
Profitability analysis
Discount vs Profit relationship
🔹 Key Insights
Sales are positively skewed (most transactions are low value)
Technology category generates highest profit
West region shows strongest performance
High discounts reduce profitability
Some products have high sales but low profit
🔹 Dashboard Features
KPI Cards (Total Sales, Total Profit, Total Orders)
Sales by Region and Category
Monthly Sales Trend
Profit Analysis
Interactive filters and drill-down
🔹 Recommendations
Optimize discount strategy
Focus on high-performing categories
Improve low-profit products
Use data-driven decision making
🔹 Conclusion

The project highlights the importance of balancing sales growth with profitability. By optimizing pricing strategies and focusing on high-performing segments, the company can improve overall performance.
